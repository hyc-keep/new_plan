# 01–04 复现执行记录

## 1. 文档用途

本文件是 01–04 阶段复现工作的持续记录。每完成一次阶段级验证或独立重复运行，都必须在这里追加一条记录，不覆盖已有记录；同时必须在对应阶段目录写入独立报告和机器可读产物，不能只在终端打印或只写本文件。

复现结果和运行资产统一写入：

```text
experiments/reproducibility_audit/
```

所有训练、测试和 runtime probe 必须使用 `--reproducibility-run`，由代码自动写入 `experiments/reproducibility_audit/repeat_runs/<logical_run_name>/`；不得通过手工路径前缀绕过输出规则。

其中：

```text
01_data_protocol/       01 数据协议验证证据
02_unet_flow/           02 UNet 流程验证证据
03_unet_stability/      03 UNet 三 seed 稳定性复现证据
04_baseline/            04 A2/B1 baseline 验证证据
repeat_runs/            独立 train→test 重复运行资产
reports/                汇总矩阵、差异报告和最终裁决
```

不得覆盖以下正式结果目录：

```text
experiments/A2_UNet_GlaS_seed3407/
experiments/A2_UNet_GlaS_seed1234/
experiments/A2_UNet_GlaS_seed2025/

```
