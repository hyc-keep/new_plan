## 2. 问题定义与动机

### 2.1 论文自己定义的问题

- 高质量 gland dense annotations 代价高，限制了全监督模型性能继续提升。
- 腺体实例分割不仅要把 gland 从背景里分出来，还要把相互接触的 glands 分开。
- 传统半监督一致性方法主要利用同一图像的不同 perturbations，未充分利用不同图像间共享的语义结构。
- 作者认为 unlabeled histology images 中跨图像的语义一致性也是可学习资源。

对应原文依据（页码）：

- `p.2-p.3`

### 2.2 核心思路（一段话概括解法方向）

- PRS2 包含两个协同模块：`S-Net` 负责监督式 gland segmentation，`PR-Net` 负责无监督的 pairwise relation learning。训练时先用标注数据初始化 `S-Net`，由于两者 encoder 共享参数，`PR-Net` 也被同步初始化；之后在 labeled + unlabeled data 上联合 fine-tune。`PR-Net` 对随机抽取的图像对建立 feature-level pairwise consistency，提升 encoder 的泛化表示；`S-Net` 则通过 `cross-entropy + Dice + object-level Dice` 多层损失学习更好地区分 touching glands。最终实现半监督 gland segmentation，并在 GlaS / CRAG 上优于多个对照方法。

关键页码：

- `p.3-p.7`

## 3. 模型/方法结构

### 3.1 总体架构

- 骨架类型：`shared-encoder semi-supervised dual-network framework`
- 主要模块：
  - `S-Net`
  - `PR-Net`
  - `PRM (Pairwise Relation Module)`
- Backbone：
  - `DeepLabv3+` as S-Net
  - encoder 为 `modified aligned Xception`
- 关键机制：
  - shared encoder
  - pairwise relation interaction
  - object-level Dice training loss

### 3.2 关键模块详细描述

**模块 1：`S-Net`**

- 位置：`监督分割主网络`
- 操作流程：
  1. 采用 `DeepLabv3+`
  2. 以 `PASCAL VOC 2012` 预训练权重初始化
  3. 替换最后 task-specific convolution layer
  4. 输出 `gland / background` 两类预测
- 页码：`p.3-p.4`

**模块 2：`Multi-level Segmentation Loss`**

- 位置：`S-Net 监督目标`
- 操作流程：
  1. `Lce` 优化像素级准确率
  2. `LDice` 优化整体 overlap
  3. `LobjDice` 约束每个 individual gland 的实例级重叠
  4. 三者相加形成 `Lseg`
- 设计理由：普通 Dice/CE 不足以处理 touching glands
- 页码：`p.4`

**模块 3：`PR-Net`**

- 位置：`无监督关系学习分支`
- 操作流程：
  1. 输入一对随机采样图像 `(xA, xB)`
  2. 通过共享 encoder 得到 `fA, fB`
  3. 送入 `PRM`
  4. 学习图像对之间的语义一致性与通用表示
- 页码：`p.4-p.5`

**模块 4：`Pairwise Relation Module (PRM)`**

- 位置：`PR-Net 核心`
- 操作流程：
  1. 计算 `fB -> fA` 的 consistency relation matrix
  2. 用 relation matrix 对对方特征做加权聚合，形成 target-highlighted feature maps
  3. 用 `smooth L1 loss` 约束原始特征与 target-highlighted 特征的一致性
- 页码：`p.4-p.5`

**模块 5：`Parameter Sharing + Two-Step Training`**

- 位置：`整体训练流程`
- 操作流程：
  1. 用 labeled set 初始化 `S-Net`
  2. 共享 encoder 参数，使 `PR-Net` encoder 同步初始化
  3. 在 labeled + unlabeled set 上联合 fine-tune 两个网络
- 页码：`p.3-p.5`

### 3.3 架构参数表（适用于基线对比论文 G 类型）

| 层/阶段 | 类型 | 通道数 | 空间尺寸 | 备注 |
|---------|------|--------|---------|------|
| S-Net encoder | `DeepLabv3+ / aligned Xception` | `未逐层明示` | patch-based | 共享给 PR-Net |
| S-Net head | segmentation classifier | `2 classes` | 同输出 | gland / background |
| PR-Net input | image pair `(xA, xB)` | `2 images` | 同 patch | 随机采样自全训练集 |
| PRM | relation matrix + weighted aggregation | `H*W by H*W` | feature space | 建模 pairwise semantic consistency |
| PR-Net loss | `smooth L1` | `N/A` | feature-level | 目标不回传梯度 |

## 4. 公式与推导

### 4.1 核心公式

公式 1：

```text
Lseg = Lce + LDice + LobjDice
```

符号说明：
- `Lce`：pixel-level cross-entropy
- `LDice`：global Dice loss
- `LobjDice`：object-level Dice loss
- 含义：多层级分割监督
- 页码：`Eq.(1), p.4`

公式 2：

```text
LobjDice =
  1/2 * [
    sum_i ( |Si| / sum_k |Sk| ) * LDice(Gi, Si)
    +
    sum_j ( |G~j| / sum_k |G~k| ) * LDice(G~j, S~j)
  ]
```

符号说明：
- `Si`：第 `i` 个预测 gland
- `Gi`：与 `Si` 最大重叠的真值 gland
- `G~j`：第 `j` 个真值 gland
- `S~j`：与 `G~j` 最大重叠的预测 gland
- 含义：从预测对象和真值对象双向衡量实例级重叠
- 页码：`Eq.(2), p.4`

公式 3：

```text
fA = F(xA; Θ),  fB = F(xB; Θ)
```

符号说明：
- `F(.)`：共享 encoder
- `Θ`：encoder 参数
- `fA, fB`：两张图像的 feature maps
- 页码：`Eq.(3), p.4`

公式 4：

```text
C_B→A = softmax( R(fA)^T · R(fB) )^T
```

符号说明：
- `R(.)`：reshape 操作，将 `H x W` 拉平
- `C_B→A`：从 `fB` 到 `fA` 的 consistency relation matrix
- 含义：衡量 feature-space 中任意两位置的语义一致性
- 页码：`Eq.(4), p.5`

公式 5：

```text
MA = R^-1( R(fB) · C_B→A )
f~A = MA + fA
f~B = MB + fB
```

符号说明：
- `MA, MB`：attention map
- `f~A, f~B`：target-highlighted feature maps
- 含义：利用另一张图像的特征对当前图像做语义增强
- 页码：`Eq.(5)-(7), p.5`

公式 6：

```text
LPR = LSL1(σ(f~A), σ(fA)) + LSL1(σ(f~B), σ(fB))
Ltotal = Lseg + alpha * LPR
```

符号说明：
- `LSL1`：smooth L1 loss
- `σ`：sigmoid
- `alpha`：无监督损失权重
- 含义：联合优化监督分割与无监督 pairwise consistency
- 页码：`Eq.(8)-(9), p.5`

### 4.2 推导过程或梯度行为（适用于 B 类损失论文）

- `f~A` 与 `f~B` 作为 target signals 时不参与反向传播。
- 作者的关键思想不是伪标签，而是通过 feature relation 自监督约束 encoder 学到更通用的 gland 语义表示。

## 8. 消融实验

### 8.1 消融设计

- 主要消融两个方面：
  - labeled data 比例变化：`20% / 50% / 100%`
  - S-Net 损失函数配置变化：
    - `LDice`
    - `Lce`
    - `Lce + LDice`
    - `Lce + LDice + LobjDice`

### 8.2 各模块贡献量化

**损失函数消融**

| Loss | GlaS Obj-D | GlaS Obj-F | GlaS Obj-H | CRAG Obj-D | CRAG Obj-F | CRAG Obj-H |
|------|------------|------------|------------|------------|------------|------------|
| `LDice` | `86.5` | `86.2` | `75.3` | `84.7` | `78.9` | `174.9` |
| `Lce + LDice` | `88.7` | `86.0` | `66.5` | `86.3` | `80.3` | `157.3` |
| `Lce + LDice + LobjDice` | `89.4` | `86.5` | `64.1` | `87.1` | `82.1` | `138.6` |

- 结论：
  - `object-level Dice loss` 明显改善 touching glands 分离
  - 在 `CRAG` 上对 `Obj-H` 的提升尤其明显

**标注比例消融**

- 论文显示：
  - `PRS2` 在 `20% / 50% / 100%` 标注比例下都稳定优于 fully-supervised `S-Net`
  - `50%` 标注的 `PRS2` 已接近 `100%` 标注的 fully-supervised `S-Net`
- 这说明 unlabeled histology data 可替代近一半 dense labels

## 11. 分类体系与研究空白（综述论文 C 类型专用）

### 11.1 论文提出的分类框架

- 本篇不是综述论文，但隐含区分了两种 SSL 思路：
  - `same-image perturbation consistency`
  - `pairwise cross-image semantic consistency`

### 11.2 论文指出的研究空白 / Open Problems

- 现有半监督方法过度依赖同图扰动一致性，没有利用不同图像间共享的 gland semantics
- gland segmentation 的半监督优化不该只追求前景/背景一致性，还要显式考虑 individual gland separation

### 11.3 对我们选题的启示

- 如果后续走半监督，你不一定非做 mean-teacher 或伪标签，也可以做 relation-level consistency

## 14. 局限性与失败案例

### 14.1 论文自述的局限性

- 当前主要验证了 gland segmentation 场景，尽管作者认为 pairwise relation SSL 具有更广泛的通用性。

### 14.2 我们观察到的潜在问题

- 需要两个网络协同训练，工程复杂度高于纯 supervised baseline
- 与 mean-teacher 类方法相比，relation matrix 的内存与实现成本更高
- 论文把另一个数据集训练集当 unlabeled，这对某些实验环境是优势，但也意味着需要额外可访问数据源

- 论文是否展示失败案例：`主要以定量/曲线为主，失败案例不突出`
- 可从结果推断的典型难点：
  - small gaps and adhesive edges

  - 数据稀缺下 representation 不稳
