`src/metrics/seg_metrics.py::compute_sample_segmentation_metrics` 曾把主表主指标 `f1` 直接赋值为像素级 Dice（`"f1": dice_score(...)`），注释 `# Dice == F1 for binary masks`。该辩护仅对像素级 F1 成立，对本项目法定口径不成立：

- plan `结直肠腺体分割_plan_优化版/01_实验执行/00_总览与规范/04_评估口径与官方脚本对齐.md` §6.3–6.4 明确规定主表 `F1` 是**对象级检测 F1**（对象匹配 overlap>0.5 → `F1_obj=2TP/(2TP+FP+FN)`），显式禁止简化成整张图像素 Dice。
- 后果：`reports/tables/unet_seed_results.csv` / `unet_mean_std_summary.csv` 的 `F1` 列与 `Dice` 列三 seed 逐位相等，F1 列失效。
