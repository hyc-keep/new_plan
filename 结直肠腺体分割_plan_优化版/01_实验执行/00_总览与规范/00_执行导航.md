# 执行导航

这份文件是 `01_实验执行` 的总入口，不是目录说明，也不是“先看一眼就可以往后跳”的提示页。

它负责先把下面六件事写死：

1. 当前项目后面到底按什么顺序推进
2. 修改任意阶段文件前，必须先重读哪些上游规范
3. 执行层当前唯一有效的主线、指标链和对外比较边界是什么
4. 各阶段什么时候允许进入下一步，什么时候必须停下回退
5. 后续每份执行文档怎样把路线层和文献层压成可实现协议
6. 这些规则最后该落到哪些代码入口、记录字段和阶段资产

换句话说，`01_实验执行` 的角色不是“项目备忘录”，而是：

> 把 `02_路线与投稿` 和 `03_文献证据` 压成一套能写代码、能跑实验、能验收、能回退的正式 protocol。

## 1. 文件角色与执行边界

当前项目已经不再处于“找方向”阶段，而是进入严格执行阶段：

- 用固定数据协议建立可复现实验链
- 用 `UNet -> ResNet34-U-Net -> LKMA -> Boundary Head -> Distance-aware Loss(可选)` 的顺序逐层验证
- 用 `GlaS` 主结果、`CRAG` 补充验证、外部对比和失败案例构成投稿证据包

这一路线不是拍脑袋决定的，而是由三组证据共同锁定：

- `02_路线与投稿/*`
- `03_文献证据/*`
- `01_实验执行` 后续各阶段正式协议

因此，后续任何代码实现、实验命名、结果表设计和阶段放行，只要与本文件冲突，都必须以后者回改到本文件。

## 6. 执行层里的四类依据

后面所有参数、协议和步骤，都必须明确属于下面四类之一：

### 6.1 官方协议固定项

这类内容可以直接写成正式协议，例如：

- `GlaS` 主 benchmark、`CRAG` 第二 benchmark
- `GlaS` 的官方 train / `TestA` / `TestB` 角色
- 主指标优先看 `F1 / Object Dice / Object Hausdorff`
- 主线先做 `UNet` 再进主 baseline

### 6.2 路线层已锁定或论文支持的候选范围

这类内容不是凭空决定，而是由路线层先锁边界、文献层给候选范围，例如：

- 输入尺寸 `512x512`
- `BCE + Dice`
- 轻量增强包
- `LKMA` 的大核候选范围
- `Boundary Head` 的边界宽度候选范围
- `lambda_dist` 的小范围候选

这类内容进入执行层后，必须继续写清：

- 当前最终选哪一个
- 相邻候选为什么不选
- 后续到哪个阶段前不得重开

### 6.3 工程默认起点

这类内容是为了让实验闭环稳定、公平、易复查而先定下来的工程协议，例如：

- `AdamW`
- `lr = 1e-3`
- `ReduceLROnPlateau`
- `epoch = 120`
- `early stopping = 20`
- 首轮 `threshold = 0.5`
- `batch size = 当前显存下稳定不溢出的最大值`

后面写文档和写代码时，最忌讳的不是使用工程起点，而是把工程起点伪装成“某篇论文唯一固定值”。

### 6.4 工程冻结规则

这类内容不来自单篇论文，但必须在执行层被写死：

- `run_name`
- 协议版本号
- checkpoint 命名
- 日志文件名
- 指标文件名
- 可视化目录
- 汇总表字段

它们当前统一以：

- `结直肠腺体分割_plan_优化版/01_实验执行/00_总览与规范/03_命名与结果记录规范.md`

## 8. 阶段推进顺序与进入条件

后面只允许按下面顺序推进：

1. 数据协议固定
2. `UNet` 单 seed 跑通
3. `UNet` 三次重复稳定
4. `ResNet34-U-Net` 三次重复稳定
5. `LKMA`
6. `Boundary Head`
7. `Distance-aware Loss`
8. 第一批外部对比
9. `CRAG` 补充验证
10. 结果汇总与投稿收口

这条顺序背后的原则固定为：

- 先证明确实能跑
- 再证明跑得稳
- 再引入唯一新增变量
- 再谈对外比较
- 最后再谈投稿强度

只要前一阶段没过线，就不允许跳到后一阶段。

### 8.1 当前明确不允许做什么

在 `UNet` 三次重复之前，不允许：

- 临时改 split
- 提前加 `LKMA`
- 提前做 `CRAG` 主表
- 提前大面积复现外部方法
